# Checkstyle & Google Java Format

## Comandos Maven

### Run application
```
mvn clean spring-boot:run 
```
### Tests
```
mvn clean test
```
### Checkstyle
```
mvn checkstyle:checkstyle
```
### Google Java Format
```
mvn com.coveo:fmt-maven-plugin:format
```
```
mvn com.coveo:fmt-maven-plugin:check
```
### Site Generation
```
mvn site
```
